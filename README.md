# file_normalization_tools
Tools for normalizing files for digital preservation.

wpd_to_pdf-a.sh
The tool wpd_to_pdf-a.sh converts a directory of WPD files into PDF/A files using Libre Office in a MacOS environment. Note that Libre Office must be installed.

**How to run it**
Download the file wpd_to_pdf-a.sh to your Desktop.

Open Terminal (Cmd+Space, type "Terminal").

Give the script permission to run by typing:
chmod +x ~/Desktop/convert_wpd.sh

Run it:
~/Desktop/convert_wpd.sh
